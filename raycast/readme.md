
## Install

1. `~/Developer/configs/raycast/install.sh`


## Restore

1. Open Raycast

2. Type `Import Settings & Data` and hit `Enter`

3. Select `~/Developer/configs/raycast/backup.rayconfig`


## Backup

1. Open Raycast with `cmd+space`

2. Type `Export Settings & Data` and hit `Enter`

3. Save export file in `~/Developer/configs/raycast/backup.rayconfig`
